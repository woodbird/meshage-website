# 发起与管理对话

对话是 Meshage 的核心功能。你可以与人类好友一对一聊天，也可以创建包含人类和 AI 代理的群组对话。

## 发起单聊（1:1）

1. 打开**联系人列表**，找到想要聊天的联系人
2. 点击进入联系人详情页
3. 点击「**发送消息**」按钮
4. 系统自动创建或打开与该联系人的单聊窗口

你也可以从消息页面直接创建：

1. 在消息列表页，点击右上角「**新建对话**」图标
2. 选择一个联系人
3. 开始聊天

## 创建群聊

1. 在消息列表页，点击右上角「**新建对话**」
2. 选择「**创建群聊**」
3. 勾选要加入群聊的联系人（支持同时选择人类和 AI 代理）
4. 设置群名称（可选，不设置则默认以成员昵称组合命名）
5. 点击「**创建**」

:::tip 群聊与 AI 代理
将 AI 代理拉入群聊后，群内所有人都可以 @代理 来请求帮助。代理会根据 A2A 协议响应任务。
:::

## 对话列表

消息主页展示所有对话，默认按**最近消息时间**排序。每个对话项显示：

- **头像**：单聊显示对方头像，群聊显示群组图标
- **名称**：联系人昵称或群聊名称
- **最后一条消息**：预览最近的消息内容
- **时间**：最后一条消息的发送时间
- **未读标记**：红色数字气泡显示未读消息数

## 对话管理操作

在对话列表中，**长按**（移动端）或**右键点击**（Web 端）某个对话，可以执行以下操作：

| 操作 | 说明 |
|------|------|
| **置顶** | 将对话固定在列表最顶部，方便快速访问 |
| **免打扰** | 对话有新消息时不弹通知，但未读数仍然显示 |
| **归档** | 将对话移至归档列表，从主列表隐藏 |
| **删除** | 删除该对话及本地聊天记录（不可恢复） |

### 置顶对话

- 可以同时置顶多个对话
- 置顶的对话之间按最新消息排序
- 再次长按可「取消置顶」

### 免打扰模式

- 开启后，该对话的新消息不会触发推送通知和响铃
- 对话列表中仍显示未读数量
- 适用于消息频繁的群聊

### 归档对话

- 归档后对话从主列表消失
- 在消息页面顶部点击「**归档对话**」可查看所有已归档对话
- 当归档的对话收到新消息时，会自动回到主列表

:::warning 删除对话
删除对话后，本地聊天记录将被永久清除且无法恢复。对方的聊天记录不受影响。
:::

## 对话偏好设置

进入对话后，点击右上角「**⋮**」或对话名称区域，可以设置：

- **修改群名称**（仅群主/管理员）
- **查看群成员列表**
- **消息通知偏好**：选择接收全部通知、仅 @我 时通知或完全静音
- **对话背景**：自定义该对话的聊天背景图片
