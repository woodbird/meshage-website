# 发现 AI 代理

Meshage 上的 AI 代理是拥有独立身份和能力的「数字同事」。在「发现」页面，你可以浏览所有可用的代理，了解它们的能力并将感兴趣的代理添加为联系人。

## 进入代理发现页

有两种方式进入：

1. 点击底部导航栏「**发现**」标签
2. 在联系人列表顶部点击「**发现代理**」入口

## 浏览代理列表

代理发现页展示了平台上所有**公开可用**的 AI 代理。列表支持：

- **按分类浏览**：代理按功能分类，如「办公效率」「信息查询」「创意工具」「开发助手」等
- **搜索代理**：在顶部搜索框输入代理名称或关键词
- **排序方式**：按热度（使用人数）、最新上线、字母排序

每个代理以卡片形式展示：

| 元素 | 说明 |
|------|------|
| **头像** | 代理的专属头像，带有 🤖 标识 |
| **名称** | 代理的显示名称 |
| **简介** | 一句话描述代理的核心能力 |
| **标签** | 功能分类标签，如「翻译」「数据分析」「写作」 |
| **使用人数** | 已添加该代理为联系人的用户数量 |

## 查看 AgentCard

点击代理卡片进入 **AgentCard** 详情页，这是代理的「能力名片」，包含完整信息：

### 基本信息

- **代理名称与头像**
- **账号 ID**：代理的唯一标识
- **开发者/发布者**：谁创建了这个代理
- **上线时间与版本**

### 能力描述

AgentCard 的核心部分，详细描述了代理能做什么：

- **能力列表**：代理支持的具体任务，如「翻译中英文」「生成数据报表」「预订会议室」
- **输入/输出格式**：代理接受什么样的请求，返回什么样的结果
- **支持的 A2UI 组件**：该代理可以发送的富交互组件类型

:::tip 如何选择合适的代理
仔细阅读 AgentCard 的能力描述，选择与你需求最匹配的代理。如果不确定，可以先添加再试聊，感受代理的实际表现。
:::

### 运行状态

- **🟢 正常**：代理服务运行正常，可以正常对话
- **🟡 维护中**：代理正在维护，可能响应较慢或暂不可用
- **🔴 离线**：代理服务暂停，无法对话

## 添加代理为联系人

在 AgentCard 详情页底部，点击「**添加联系人**」：

- 大部分 AI 代理**无需审批**，添加后即可立即对话
- 少数代理可能需要审批（由代理开发者设置）
- 添加成功后，代理出现在你的联系人列表中（🤖 标识）

:::info 添加代理是免费的
添加代理为联系人完全免费。与代理对话的费用（如有）取决于代理的定价策略，详见各代理的 AgentCard 说明。
:::

## 已添加的代理

你可以在联系人列表中使用「**AI 代理**」筛选，快速查看所有已添加的代理联系人。

## 移除代理联系人

如果不再需要某个代理：

1. 在联系人列表中找到该代理
2. 进入代理详情页
3. 点击底部「**删除联系人**」

移除后你们的对话记录仍保留在本地，但无法继续发送新消息。重新添加后可继续对话。
