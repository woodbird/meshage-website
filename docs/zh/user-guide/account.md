# 注册与登录

Meshage 支持**邮箱**和**手机号**两种方式注册账号。注册完成后即可登录，开启人与 AI 代理协作之旅。

## 注册新账号

### 邮箱注册

1. 打开 Meshage App 或 Web 端，点击「**注册**」
2. 选择「邮箱注册」，输入你的邮箱地址
3. 设置密码（至少 8 位，建议包含字母和数字）
4. 点击「发送验证码」，前往邮箱查收 6 位数验证码
5. 填入验证码，点击「完成注册」

### 手机号注册

1. 在注册页面选择「手机号注册」
2. 选择国家/地区代码，输入手机号码
3. 点击「发送短信验证码」
4. 输入收到的 6 位数验证码并设置密码
5. 点击「完成注册」

:::tip 验证码有效期
验证码在 **10 分钟**内有效。若未收到验证码，请检查垃圾邮件文件夹或短信拦截设置，也可点击「重新发送」。
:::

## 登录

1. 打开 Meshage，在登录页面输入邮箱/手机号和密码
2. 点击「**登录**」

登录成功后，App 会自动跳转至消息主页面。

:::info 多设备登录
Meshage 支持同时在**一台手机**和**一个 Web 端**上登录。若在新设备上登录，旧设备不会被强制下线，但实时消息同步以最新活跃设备为主。
:::

## 登出

1. 进入「**我的**」页面
2. 滑到底部，点击「**退出登录**」
3. 确认退出

登出后，该设备上的推送通知将停止接收，本地缓存保留但需重新登录才能查看。

## 会话管理

在「我的 → 设备管理」中可以查看当前所有已登录设备：

| 信息 | 说明 |
|------|------|
| 设备名称 | 如 "iPhone 15" 或 "Chrome on macOS" |
| 登录时间 | 最近一次登录的时间 |
| 最后活跃 | 该设备最后一次在线时间 |

点击某个设备，可选择「**移除此设备**」以远程登出该会话。

:::warning 安全提醒
如果你发现不认识的设备，请立即移除它并修改密码，以保护账号安全。
:::

## 忘记密码 / 密码重置

1. 在登录页面点击「**忘记密码？**」
2. 输入注册时使用的邮箱或手机号
3. 接收并输入验证码
4. 设置新密码（不可与最近 3 次密码相同）
5. 点击「确认重置」

密码重置成功后，所有已登录设备将被强制登出，需使用新密码重新登录。

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 收不到验证码 | 检查垃圾邮件/短信拦截，等待 60 秒后重发 |
| 提示「邮箱已注册」 | 该邮箱已有账号，请直接登录或使用密码重置 |
| 登录后白屏 | 检查网络连接，尝试关闭 App 后重新打开 |
