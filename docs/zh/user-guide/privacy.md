# 隐私设置

Meshage 提供灵活的隐私控制，让你决定谁可以找到你、谁可以直接添加你为联系人。

## 进入隐私设置

1. 点击底部导航栏「**我的**」
2. 点击「**设置**」→「**隐私**」

## 联系人审批开关（require_approval）

此开关决定其他用户添加你为联系人时，是否需要你手动审批。

| 状态 | 行为 |
|------|------|
| **开启**（默认） | 他人发来的联系人请求进入「待审批」列表，你需要手动同意或拒绝 |
| **关闭** | 任何人向你发送联系人请求时自动通过，直接成为联系人 |

:::tip 推荐设置
建议保持**开启**状态。这样你可以过滤不认识的人，保护自己的通讯录干净整洁。
:::

修改方式：在隐私设置页找到「**联系人审批**」，切换开关即可。

## 口令免审批（Passphrase）

即使你开启了联系人审批，也可以设置一个**口令**让信任的朋友免审批直接添加你。

- 口令长度：**4 ～ 32 个字符**
- 支持字母、数字和常见符号
- 可以随时修改或清空

:::info 口令工作原理
当对方添加你时，如果填入了正确的口令，联系人请求会**自动通过**，无需你手动审批。口令不正确的请求仍然进入审批队列。
:::

详细的口令使用指南见：[口令免审批](/zh/user-guide/passphrase)

## 搜索可见性

控制其他用户是否能通过搜索找到你：

| 选项 | 说明 |
|------|------|
| **所有人可搜索**（默认） | 任何人都可以通过你的账号 ID 或昵称搜索到你 |
| **仅限账号 ID** | 只有输入你的完整账号 ID 才能找到你，昵称搜索不显示 |
| **不可搜索** | 你的资料不出现在任何搜索结果中，只能通过分享名片添加 |

:::warning 隐身后的影响
选择「不可搜索」后，新朋友只能通过你主动分享的名片二维码或链接添加你。已有联系人不受影响。
:::

## 在线状态

你可以选择是否向其他人展示你的在线/离线状态：

- **显示在线状态**（默认）：联系人可以看到你是否在线
- **隐藏在线状态**：你的状态始终显示为离线，但你可以正常收发消息

## 已读回执

控制你是否向对方发送消息已读状态：

- **开启**（默认）：对方可以看到你已读了他的消息
- **关闭**：对方无法得知你是否已读

关闭已读回执后，你也将无法看到对方的已读状态（双向生效）。

## 隐私设置总览

| 设置项 | 默认值 | 位置 |
|--------|--------|------|
| 联系人审批 | 开启 | 隐私 → 联系人审批 |
| 口令免审批 | 未设置 | 隐私 → 口令免审批 |
| 搜索可见性 | 所有人可搜索 | 隐私 → 搜索可见性 |
| 在线状态 | 显示 | 隐私 → 在线状态 |
| 已读回执 | 开启 | 隐私 → 已读回执 |
