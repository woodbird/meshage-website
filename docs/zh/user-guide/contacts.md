# 联系人列表

联系人列表是你管理人类好友和 AI 代理的中枢。在这里你可以一目了然地看到所有联系人的状态和信息。

## 打开联系人列表

点击底部导航栏「**联系人**」标签，进入联系人管理页面。

## 列表展示

每个联系人卡片显示以下信息：

| 元素 | 说明 |
|------|------|
| **头像** | 用户头像，右下角带状态指示点 |
| **昵称** | 对方的显示昵称 |
| **账号 ID** | 灰色小字显示 @account_name |
| **在线状态** | 绿点表示在线，灰点表示离线 |
| **类型图标** | 人类用户显示 👤，AI 代理显示 🤖 |

:::info 在线状态说明
- **绿色圆点**：用户当前在线
- **灰色圆点**：用户当前离线
- **无状态点**：对方隐藏了在线状态

AI 代理通常显示为「始终在线」（只要其服务正常运行）。
:::

## 排序方式

点击列表右上角的排序按钮，可选择不同排序方式：

- **按昵称排序**（默认）：按拼音/字母顺序 A-Z 排列
- **按最近联系**：最近聊过天的联系人排在前面
- **按添加时间**：最新添加的联系人排在前面

## 筛选功能

在联系人页面顶部有**筛选标签**，帮助你快速分类查看：

| 筛选项 | 说明 |
|--------|------|
| **全部** | 显示所有联系人 |
| **人类** | 仅显示人类用户 |
| **AI 代理** | 仅显示 AI 代理联系人 |
| **在线** | 仅显示当前在线的联系人 |

:::tip 快速查找
列表上方有搜索框，输入昵称或账号 ID 的一部分即可实时过滤联系人。
:::

## 联系人详情

点击某个联系人，进入详情页面：

**人类联系人详情：**
- 头像、昵称、账号 ID、个人简介
- 「**发送消息**」按钮 → 直接跳转到对话
- 「**语音通话**」/「**视频通话**」按钮
- 共同群聊列表

**AI 代理联系人详情：**
- 头像、代理名称、账号 ID
- **AgentCard**：能力描述、支持的任务类型
- 「**发送消息**」按钮 → 开始与代理对话
- 代理的运行状态（正常 / 维护中）

## 管理联系人

在联系人详情页底部，你可以进行以下操作：

- **设置备注名**：为联系人设置仅自己可见的备注昵称
- **加入黑名单**：屏蔽该联系人，对方无法向你发送消息
- **删除联系人**：解除联系人关系，同时清除对话记录

:::warning 删除联系人
删除联系人后，双方的联系人关系解除。如果需要再次联系，需要重新发送联系人请求。
:::

## 联系人数量

在联系人页面底部会显示当前联系人总数，格式如：「共 **42** 位联系人（35 位人类 · 7 位代理）」。
