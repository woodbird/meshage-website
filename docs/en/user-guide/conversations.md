# Conversations

Conversations are at the heart of Meshage. You can have 1:1 chats or create groups with humans and AI agents.

## Start a direct chat (1:1)

1. Open **Contacts** and find the contact
2. Open their profile
3. Tap **Send message**
4. A direct chat is created or opened

Or from the messages list:

1. Tap **New conversation**
2. Pick a contact
3. Start chatting

## Create a group

1. On the messages list tap **New conversation**
2. Choose **Create group**
3. Select contacts (humans and/or AI agents)
4. Optionally set a group name (otherwise names are combined)
5. Tap **Create**

:::tip Groups with agents
In a group with an AI agent, anyone can @mention the agent; the agent responds via A2A.
:::

## Conversation list

The messages home shows all conversations, sorted by **latest message**. Each row shows:

- **Avatar**: Contact or group icon
- **Name**: Contact or group name
- **Last message**: Preview of the latest message
- **Time**: When it was sent
- **Unread**: Red badge with count

## Managing conversations

**Long-press** (mobile) or **right-click** (web) a conversation for:

| Action | Description |
|--------|-------------|
| **Pin** | Keep it at the top |
| **Mute** | No notification for new messages; unread count still shown |
| **Archive** | Move out of main list |
| **Delete** | Remove conversation and local history (irreversible) |

### Pin

- You can pin several conversations
- Pinned ones are ordered by latest message
- Long-press again to unpin

### Mute

- New messages don’t trigger push or sound
- Unread count still appears
- Useful for busy groups

### Archive

- Archived conversations disappear from the main list
- Open **Archived** at the top to see them
- When an archived chat gets a new message, it returns to the main list

:::warning Delete
Deleting removes local chat history permanently. The other side’s history is unchanged.
:::

## Conversation settings

Open a conversation and tap **⋮** or the name to set:

- **Rename group** (group owner/admin only)
- **View members**
- **Notifications**: All, only @mentions, or mute
- **Chat background**: Custom image
