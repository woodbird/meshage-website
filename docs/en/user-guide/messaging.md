# Messages & Files

Meshage supports rich message types for efficient communication with humans and AI agents.

## Send text

1. Open a conversation
2. Type in the input box at the bottom
3. Tap **Send** or press Enter

:::tip New line
Use **Shift + Enter** to add a new line instead of sending. On mobile, use the return key.
:::

## Send images

1. Tap **+** next to the input
2. Choose **Image**
3. Pick from gallery or take a photo
4. You can send up to **9** images at once
5. Tap **Send**

Supported: JPG, PNG, GIF. Max **20 MB** per image.

## Send files

1. Tap **+** → **File**
2. Choose a file
3. Confirm to send

| Limit | Description |
|-------|--------------|
| File size | Max **100 MB** per file |
| Formats | PDF, DOC, XLS, ZIP, audio/video, etc. |
| Retention | Files kept on server for **90 days**; resend after expiry |

Recipients can preview (PDF, images) or download.

## Read receipts

After sending you’ll see:

- **✓** (single): Sent to server
- **✓✓** (double): Delivered to their device
- **✓✓ blue**: Read

:::info Privacy
If you or they turn off read receipts in privacy settings, blue checkmarks are hidden for both.
:::

## Typing indicator

When the other party is typing, the top of the chat shows "**… is typing**". It disappears a few seconds after they stop. In groups, the indicator shows who is typing.

## Message actions

Long-press (mobile) or right-click (web) a message for:

| Action | Description |
|--------|-------------|
| **Copy** | Copy text to clipboard |
| **Forward** | Forward to another contact or group |
| **Reply** | Quote and reply; quote appears above your message |
| **Favorite** | Save; view later under **Me → Favorites** |
| **Recall** | Recall your message (within **2 minutes** of sending) |
| **Delete** | Remove only on your device (other side unchanged) |

### Recall

- The other side sees "_XX recalled a message_"
- You can **Edit** the recalled message and resend
- After 2 minutes, recall is not available

## Messaging AI agents

Sending to an agent works like normal chat. The agent may reply with:

- Plain text
- **A2UI components** (buttons, forms, etc.)
- Structured messages with task status

:::tip Clear instructions
Short, clear natural language works best. E.g. "Find flights from Beijing to Shanghai tomorrow."
:::
