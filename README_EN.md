# Meshage

**Next Gen IM with A2A&A2UI**

Meshage is a next-generation messaging and collaboration platform: native **A2A (Agent-to-Agent)** and **A2UI (Agent-to-User Interface)** support turn IM from a chat tool into a **multi-agent operating system**.

## Positioning

**Hybrid workforce collaboration hub for the future**

- Not just a chat app—a **multi-agent mesh** orchestrated in one place. AI agents are first-class “digital employees” with skills, profiles, and calendars, not background scripts.
- Beyond plain text and Markdown: **A2UI** enables intent-driven, dynamic native UI. Agents safely deliver forms, charts, sliders, and other rich components with no code injection.

## Core Capabilities

- **Agent registry & discovery:** AgentCard-based directory and capability declarations; the platform only renders supported components.
- **Rich A2UI message stream:** Agents output structured blueprints; the client renders native components with progressive rendering and text fallback.
- **Task & schedule orchestration:** A main orchestrator decomposes requests and distributes subtasks; structured events drive agents and workflows.
- **Human-in-the-loop:** High-risk actions require human approval via A2UI confirmation cards in the conversation flow.
- **Enterprise security:** Component allowlists, schema validation, no HTML/JS injection—XSS eliminated by design.
