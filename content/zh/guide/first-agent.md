# 与 AI 代理对话

Meshage 的一大特色是**与 AI 代理协作**。AI 代理不仅能与你文字对话，还能发送按钮、表单、滑块等富交互组件（A2UI），让任务完成更高效、更直观。

## 第一步：进入代理列表

1. 打开 Meshage App 或网页版
2. 点击底部导航栏的「**代理**」或「**AI 代理**」图标
3. 进入可用的 AI 代理列表页面

## 第二步：浏览并添加代理

1. 浏览列表中已上线的 AI 代理
2. 每个代理都有简介和能力说明（如「会议预订」「报表生成」等）
3. 找到你感兴趣的代理，点击「**添加为联系人**」或「**开始对话**」
4. 若需添加为联系人，请求通常会自动通过（代理无需人工审批）

## 第三步：打开对话并互动

1. 在代理列表或联系人中选择该代理，进入会话
2. 发送消息，例如：「帮我预订明天下午 2 点的会议室」
3. 代理会回复，并可能附带 **A2UI 富交互组件**：
   - **按钮**：如「确认」「取消」「重选」
   - **表单**：填写日期、人数、偏好等
   - **滑块 / 选择器**：调整预算、时间范围等参数

:::info 认识 A2UI 组件
**A2UI**（Agent-to-User Interface）是 Meshage 的核心能力之一。代理发送的不再是单调的长文本，而是可点击、可填写的原生 UI 组件。你只需在组件上操作，即可完成任务，无需复制粘贴或来回切换应用。
:::

## 第四步：处理任务状态与审批

- **任务状态**：代理在处理复杂任务时，可能会显示「处理中」「已完成」等状态提示
- **审批请求**：当代理执行高风险操作（如修改数据、发送部署）时，会在聊天中出现**审批卡片**
  - 卡片中会说明操作内容和影响
  - 你需要点击「**批准**」或「**拒绝**」，代理才会继续执行
  - 这是 Meshage 的「人在回路」设计，确保关键操作由人工把关

## 与人类聊天的区别

| 能力 | 人类联系人 | AI 代理 |
|------|------------|---------|
| 文字消息 | ✅ | ✅ |
| 文件 / 图片 | ✅ | ✅ |
| A2UI 富组件 | ❌ | ✅ |
| 任务状态指示 | ❌ | ✅ |
| 审批确认卡片 | ❌ | ✅（高风险操作） |

## 常见 AI 代理场景示例

- **会议预订**：选择日期、时间、会议室，代理返回确认卡片
- **报表生成**：描述需求，代理生成图表并附带筛选器、导出按钮
- **日程管理**：通过表单添加日程，代理同步到日历
- **客服问答**：提问后，代理可能返回选项按钮供你快速选择
