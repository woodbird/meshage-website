# 下载与安装

Meshage 支持多端使用，你可以根据自己的设备选择最合适的方式开始体验。

## 系统要求

| 平台 | 最低版本要求 |
|------|--------------|
| **iOS** | iOS 15.0 及以上 |
| **Android** | Android 8.0（API 26）及以上 |
| **Web 浏览器** | Chrome 90+、Safari 14+、Edge 90+ 等现代浏览器 |

:::tip 快速体验推荐
**最快上手方式**：直接在浏览器中打开 Meshage 网页版，无需下载 App。访问 [meshage.io](https://meshage.io) 即可开始使用。
:::

## 方式一：iOS App Store

1. 打开 **App Store**
2. 搜索「**Meshage**」
3. 点击「获取」或「下载」
4. 安装完成后，点击图标打开 Meshage

## 方式二：Android

### Google Play 商店（推荐）

1. 打开 **Google Play 商店**
2. 搜索「**Meshage**」
3. 点击「安装」
4. 安装完成后，在应用列表中打开 Meshage

### APK 直接安装

若无法使用 Google Play，可从 Meshage 官网下载 APK 安装包：

1. 访问 [meshage.io/download](https://meshage.io/download)
2. 选择「Android APK」下载
3. 下载完成后，打开 APK 文件并按提示完成安装
4. 若系统提示「未知来源」，需在设置中允许安装来自此来源的应用

:::warning 安全提示
请务必从 Meshage 官方渠道下载 App，勿使用第三方来源的安装包，以免造成账号或数据风险。
:::

## 方式三：Web 网页版

1. 使用任意现代浏览器访问 [meshage.io](https://meshage.io)
2. 点击「**立即使用**」或「**Web 版登录**」
3. 使用已有账号登录，或按指引完成注册
4. 开始使用

Web 版功能与 App 基本一致，适合在电脑上办公使用，也便于快速体验平台功能。

## 安装后首次启动

- **App 版**：首次打开会显示欢迎页或登录界面，按提示注册或登录
- **Web 版**：首次访问会跳转到登录/注册页，完成账号创建后可保存登录状态

若遇到无法安装、闪退或网页无法加载等问题，请检查网络连接，或参考 [常见问题（FAQ）](/zh/faq/) 寻求帮助。
