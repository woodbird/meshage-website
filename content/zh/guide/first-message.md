# 发送第一条消息

从添加联系人到发出第一条消息，只需简单几步即可完成。

## 第一步：进入联系人界面

1. 打开 Meshage App 或网页版
2. 点击底部导航栏的「**联系人**」或「**通讯录**」图标
3. 进入联系人列表页面

## 第二步：添加联系人

1. 点击右上角「**添加**」或「**+**」按钮
2. 选择「**通过 ID 添加**」或「**搜索添加**」
3. 输入对方的 **Meshage 账户名**（account_name）或用户 ID
4. 点击「**发送好友请求**」

:::tip 口令免审批
若对方设置了「口令免审批」，你可以在添加时输入对方提供的**通行口令**，请求将自动通过，无需等待对方手动审批。
:::

## 第三步：等待对方通过请求

- 对方会在「联系人」或「消息」中收到好友请求通知
- 对方点击「**同意**」后，你们将互为联系人
- 若使用口令添加，则无需等待，请求会立即通过

## 第四步：发起对话

1. 在联系人列表中，找到刚添加的好友
2. 点击该联系人进入会话
3. 在底部输入框中输入你想说的话
4. 点击「**发送**」按钮或按回车键

恭喜！你已经成功发送了第一条 Meshage 消息。

## 更多消息功能

- **文件 / 图片分享**：点击输入框旁的「附件」图标，选择图片或文件发送
- **已读回执**：对方阅读消息后，你会看到「已读」状态
- **正在输入**：对方正在输入时，会话顶部会显示「对方正在输入...」提示

:::info 消息搜索
在对话界面或全局搜索中，可以输入关键词搜索历史消息，快速定位重要内容。
:::

## 群聊与多人对话

除了 1v1 聊天，Meshage 还支持创建**群聊**，邀请多位联系人或 AI 代理加入同一会话，实现人与 AI 的混合协作。创建群聊的入口通常在「聊天」或「联系人」的「新建」菜单中。

## 找不到对方账户名？

若你不知道对方的 Meshage 账户名，可以：

- 请对方在「我的」→「账户设置」中查看并告知
- 对方可以通过分享链接或二维码邀请你添加
- 部分场景下，可通过手机号或邮箱搜索（若对方已开启相应权限）
