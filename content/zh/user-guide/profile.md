# 编辑个人资料

个人资料是其他用户和 AI 代理认识你的第一张名片。完善资料可以让好友更容易找到你。

## 进入资料编辑页

1. 点击底部导航栏「**我的**」
2. 点击页面顶部的头像区域或「**编辑资料**」按钮

## 头像

点击头像区域即可更换头像：

- **从相册选择**：选取手机相册中的照片，支持裁剪为圆形
- **拍照**：直接使用摄像头拍摄
- **Web 端**：点击后弹出文件选择器，选择本地图片

:::tip 头像规格
推荐使用 **300×300 像素以上** 的正方形图片。支持 JPG、PNG 格式，文件大小不超过 **5 MB**。
:::

上传成功后，新头像会在所有设备上同步更新。

## 显示昵称

显示昵称是其他用户在聊天中看到的名字：

- 长度限制：**1 ～ 30 个字符**
- 支持中文、英文、数字和常见符号
- 可以随时修改，即时生效

## 账号 ID（account_name）

账号 ID 是你在 Meshage 上的**唯一标识符**，其他人可以通过它精准找到你。

| 规则 | 说明 |
|------|------|
| 格式要求 | 仅限字母、数字和下划线，以字母开头 |
| 长度范围 | 4 ～ 24 个字符 |
| 唯一性 | 全平台唯一，不可与他人重复 |
| 修改频率 | 每 **30 天**仅可修改一次 |

:::warning 修改限制
账号 ID 修改后，旧 ID 立即释放，30 天内不可再次更改。请谨慎设置，确保你对新 ID 满意后再提交。
:::

修改步骤：

1. 在编辑资料页找到「**账号 ID**」
2. 输入新的 ID，系统实时检查是否可用
3. 确认无误后点击「保存」

## 个人简介

简介会显示在你的个人主页和联系人详情中：

- 长度限制：**最多 200 个字符**
- 支持 emoji 表情
- 留空则不显示

好的简介可以帮助别人快速了解你。例如：

> 🎨 设计师 | 咖啡爱好者 | 用 Meshage 和 AI 一起工作

## 保存与同步

修改任何资料后，点击页面右上角「**保存**」按钮。保存成功后：

- 所有已登录设备上的资料**实时同步**
- 对方在聊天和联系人列表中看到的信息会即时更新
- 已发送的历史消息中的头像和昵称也会同步刷新

:::info 修改不可用？
如果「账号 ID」字段显示为灰色不可编辑，说明距上次修改不足 30 天。你可以查看字段下方提示的「下次可修改时间」。
:::

## 预览个人主页

在编辑资料页底部，点击「**预览个人主页**」可以看到其他用户搜索到你时看到的样子，包括头像、昵称、账号 ID 和简介。
