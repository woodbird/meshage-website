# 发送消息与文件

Meshage 支持丰富的消息类型，让你与人类好友和 AI 代理高效沟通。

## 发送文本消息

1. 进入对话窗口
2. 在底部输入框中输入文字
3. 点击「**发送**」按钮或按 Enter 键

:::tip 换行
在输入框中按 **Shift + Enter** 可以换行，而不是发送消息。移动端点击键盘上的换行键即可。
:::

## 发送图片

1. 点击输入框左侧的「**＋**」按钮
2. 选择「**图片**」
3. 从相册选取或直接拍照
4. 可选择多张图片同时发送（最多 9 张）
5. 点击「发送」

支持 JPG、PNG、GIF 格式，单张图片不超过 **20 MB**。

## 发送文件

1. 点击「**＋**」→「**文件**」
2. 选择本地文件
3. 确认后发送

| 限制 | 说明 |
|------|------|
| 单文件大小 | 最大 **100 MB** |
| 支持格式 | PDF、DOC、XLS、ZIP、音视频等常见格式 |
| 保存期限 | 文件在服务器保留 **90 天**，过期后需重新发送 |

接收方点击文件消息可预览（PDF、图片）或下载到本地。

## 已读回执

发送消息后，你可以看到消息的送达和已读状态：

- **✓**（单勾）：消息已发送到服务器
- **✓✓**（双勾）：消息已送达对方设备
- **✓✓ 蓝色**（蓝色双勾）：对方已阅读

:::info 已读回执与隐私
如果你或对方在隐私设置中关闭了已读回执，蓝色双勾将不会显示。双方的已读状态同时隐藏。
:::

## 正在输入指示器

当对方正在输入时，对话顶部会显示「**对方正在输入...**」的提示。这个指示器会在对方停止输入几秒后自动消失。

群聊中会显示具体是哪位成员正在输入。

## 消息操作

长按（移动端）或右键点击（Web 端）某条消息，可执行：

| 操作 | 说明 |
|------|------|
| **复制** | 复制消息文本到剪贴板 |
| **转发** | 将消息转发给其他联系人或群聊 |
| **回复** | 引用该消息并进行回复，引用内容会附在新消息上方 |
| **收藏** | 收藏消息，稍后在「我的 → 收藏」中查看 |
| **撤回** | 撤回自己发送的消息（发送后 **2 分钟**内有效） |
| **删除** | 仅删除本地的该条消息（对方不受影响） |

### 消息撤回

- 撤回后对方会看到「_XX 撤回了一条消息_」的提示
- 撤回的消息可以「**重新编辑**」
- 超过 2 分钟的消息无法撤回

## 发送给 AI 代理

向 AI 代理发消息的体验与普通聊天一致。代理处理消息后，可能会返回：

- 纯文本回复
- **A2UI 富交互组件**（按钮、表单等）
- 带有任务状态的结构化消息

:::tip 给代理发指令
与 AI 代理聊天时，使用清晰简洁的自然语言描述你的需求，效果最佳。例如「帮我查一下明天北京到上海的航班」。
:::
