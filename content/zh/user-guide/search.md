# 消息搜索

随着使用时间增长，聊天记录会越来越多。Meshage 的消息搜索功能帮助你快速定位历史对话中的关键内容。

## 全局搜索

在消息列表页面顶部的**搜索栏**中输入关键词，即可跨所有对话搜索消息。

1. 点击消息页面顶部的搜索框
2. 输入要搜索的关键词
3. 系统自动显示匹配结果

搜索结果按**对话分组**展示，每个对话下显示匹配的消息数量和最近的匹配片段。

:::tip 搜索技巧
- 使用精确短语搜索更准确，如「项目报告」而非「项目」
- 搜索支持中英文混合
- 搜索结果中关键词会**高亮**显示
:::

## 对话内搜索

如果你知道消息在哪个对话中，可以在该对话内搜索：

1. 进入目标对话
2. 点击右上角「**⋮**」→「**搜索消息**」
3. 输入关键词
4. 结果按时间倒序排列
5. 点击某条结果，自动跳转到该消息在对话中的位置

## 筛选条件

在搜索结果页面，你可以使用筛选器缩小范围：

### 按联系人筛选

点击搜索结果页的「**联系人**」筛选按钮：

- 选择特定联系人，只显示与该人相关的消息
- 支持选择多个联系人
- 同时搜索该联系人发送和接收的消息

### 按日期筛选

点击「**日期**」筛选按钮：

| 快捷选项 | 范围 |
|----------|------|
| 今天 | 当天的消息 |
| 最近 7 天 | 过去一周内 |
| 最近 30 天 | 过去一个月内 |
| 自定义范围 | 选择开始日期和结束日期 |

:::info 联合筛选
联系人筛选和日期筛选可以**同时使用**，进一步缩小搜索范围。例如：搜索「报告」+ 联系人「小李」+ 最近 7 天。
:::

## 搜索结果操作

找到目标消息后，你可以：

- **点击跳转**：直接跳转到该消息在原对话中的位置，前后上下文一目了然
- **长按操作**：对搜索结果执行复制、转发、收藏等消息操作

## 搜索支持的内容类型

| 类型 | 是否可搜索 |
|------|------------|
| 文本消息 | ✅ 全文搜索 |
| 文件名 | ✅ 按文件名搜索 |
| 图片 | ❌ 暂不支持按图片内容搜索 |
| A2UI 组件 | ✅ 搜索组件中的文本内容 |

:::warning 搜索限制
消息搜索基于本地缓存和服务端索引。如果你清除了本地数据，较旧的消息可能需要联网后才能搜索到。
:::

## 常见搜索场景

- **找到某人发过的文件**：搜索文件名关键词 + 联系人筛选
- **回顾某次讨论**：搜索讨论主题关键词 + 日期范围
- **查找代理返回的结果**：搜索任务相关关键词 + 选择对应的 AI 代理
