# 主题与显示

Meshage 提供多种主题模式，你可以根据个人偏好和使用环境选择最舒适的视觉风格。

## 进入主题设置

1. 点击底部导航栏「**我的**」
2. 进入「**设置**」→「**主题与显示**」

## 主题模式

Meshage 支持三种主题模式：

### 浅色模式（Light Mode）

- 白色/浅灰色背景，深色文字
- 适合在**光线充足**的环境中使用
- 界面明亮清爽

### 深色模式（Dark Mode）

- 深色/黑色背景，浅色文字
- 适合在**夜间或暗光**环境中使用
- 减少屏幕亮度对眼睛的刺激
- OLED 屏幕设备可节省电量

### 跟随系统（Follow System）

- App 的主题自动跟随操作系统的深浅色设置
- 当系统切换为深色模式时，App 自动切换
- 适合已设置了系统级日夜模式切换的用户

:::tip 推荐设置
选择「**跟随系统**」最为省心。大多数手机支持按时间自动切换深浅色模式（如日落后切换为深色），App 会自动适配。
:::

## 切换主题

1. 在「主题与显示」页面找到「**主题模式**」
2. 选择「浅色」「深色」或「跟随系统」
3. 选择后**立即生效**，无需重启

## 主题影响的范围

切换主题后，以下界面元素会同步变化：

| 元素 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景色 | 白色/浅灰 | 深灰/纯黑 |
| 文字色 | 深色 | 浅色/白色 |
| 消息气泡 | 浅蓝/白色气泡 | 深蓝/深灰气泡 |
| 导航栏 | 白色底 | 深色底 |
| A2UI 组件 | 浅色主题渲染 | 深色主题渲染 |
| 图标 | 深色图标 | 浅色图标 |

:::info A2UI 组件的主题适配
AI 代理发送的 A2UI 组件会自动适配当前主题。组件在深色模式下会使用深色配色方案，保持视觉一致性。
:::

## 字体大小

在「主题与显示」页面中还可以调整文字大小：

| 选项 | 说明 |
|------|------|
| **小** | 紧凑布局，适合信息密集场景 |
| **标准**（默认） | 平衡的阅读体验 |
| **大** | 更大的文字，方便阅读 |
| **特大** | 最大字体，适合视力不便的用户 |

调整字体大小会影响所有界面文字，包括聊天消息、菜单文字和按钮文字。

:::tip 预览效果
调整字体大小时，页面底部会实时显示预览效果，帮助你选择最合适的大小。
:::

## 聊天背景

除了全局主题，你还可以为每个对话设置独立的聊天背景：

1. 进入对话
2. 点击右上角「**⋮**」→「**聊天背景**」
3. 选择预设背景或从相册选择自定义图片

聊天背景仅在对应对话中显示，不影响其他对话和界面。

## 多设备独立设置

与语言设置类似，主题设置在每台设备上是**独立**的：

- 手机上使用深色模式，Web 端使用浅色模式
- 切换某台设备的主题不会影响其他设备
- 适合在不同设备上使用不同的视觉偏好

## 无障碍支持

Meshage 在主题设计上兼顾了无障碍需求：

- 所有主题模式均满足 **WCAG 2.1 AA** 级别的对比度要求
- 支持系统级的辅助功能设置（如粗体文本、高对比度）
- 关键操作区域有足够的点击区域大小
