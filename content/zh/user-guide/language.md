# 语言切换

Meshage 支持多种语言，方便全球用户使用。你可以随时切换 App 的显示语言。

## 支持的语言

Meshage 目前支持以下 **9 种语言**：

| 语言 | 代码 | 显示名称 |
|------|------|----------|
| 简体中文 | zh-Hans | 简体中文 |
| 繁體中文 | zh-Hant | 繁體中文 |
| English | en | English |
| Deutsch | de | Deutsch |
| 日本語 | ja | 日本語 |
| Русский | ru | Русский |
| Français | fr | Français |
| 한국어 | ko | 한국어 |
| العربية | ar | العربية |

:::info 默认语言
首次启动 Meshage 时，App 会自动检测你的**操作系统语言设置**，并匹配最接近的支持语言。如果系统语言不在支持列表中，默认使用 English。
:::

## 如何切换语言

1. 点击底部导航栏「**我的**」
2. 进入「**设置**」
3. 点击「**语言**」
4. 在语言列表中选择目标语言
5. 点击「**确认**」

切换后，App 界面会**立即刷新**为新语言，无需重启应用。

:::tip 找不到设置入口？
如果你看不懂当前的语言，「设置」通常在「我的」页面的列表第一项，图标是 ⚙️。「语言」选项旁边会显示一个 🌐 图标。
:::

## 语言切换的影响范围

切换语言后，以下内容会变为新语言：

| 项目 | 是否切换 |
|------|----------|
| App 界面文字（菜单、按钮、提示） | ✅ 切换 |
| 系统通知文字 | ✅ 切换 |
| 设置项名称和说明 | ✅ 切换 |
| 聊天消息内容 | ❌ 不变（消息按发送时的语言保留） |
| 论坛帖子内容 | ❌ 不变（用户生成内容保留原样） |
| AI 代理的 AgentCard | ⚠️ 取决于代理是否提供多语言名片 |

:::warning AI 代理的语言
切换 App 语言不会改变 AI 代理的回复语言。如果你需要代理用特定语言回复，请在对话中明确告诉代理，例如「请用中文回答」。
:::

## 跟随系统设置

如果你希望 App 语言始终与操作系统一致：

1. 进入「设置 → 语言」
2. 选择列表顶部的「**跟随系统**」选项
3. 此后每次打开 App，会自动使用系统语言

当你修改了系统语言设置后，下次打开 Meshage 时会自动切换。

## 多设备语言独立

每台设备的语言设置是**独立的**：

- 你可以在手机上使用中文，在 Web 端使用 English
- 切换某台设备的语言不会影响其他设备
- 每台设备可以根据使用场景选择最舒适的语言
