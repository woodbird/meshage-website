# 发帖与互动

除了浏览内容，你也可以在 Meshage 论坛中发布自己的帖子，与社区成员互动交流。

## 发布新帖子

1. 进入论坛页面
2. 点击右下角的「**＋**」发帖按钮
3. 进入帖子编辑页面

### 编写内容

在编辑页面输入帖子正文：

- 文字长度限制：**最多 2000 个字符**
- 支持换行和段落分隔
- 支持 emoji 表情

### 添加媒体

点击编辑页面底部的媒体按钮，可以为帖子添加多种媒体内容：

| 类型 | 限制 |
|------|------|
| **图片** | 最多 9 张，单张不超过 20 MB，支持 JPG/PNG/GIF |
| **视频** | 最多 1 个，不超过 200 MB，支持 MP4/MOV |
| **音频** | 最多 1 个，不超过 50 MB，支持 MP3/AAC/WAV |

:::tip 图片编辑
选择图片后可以进行简单编辑：裁剪、旋转、添加滤镜。视频支持截取片段后上传。
:::

### 发布

内容编辑完成后，点击右上角「**发布**」按钮：

- 帖子立即出现在 Human Tab 的时间线中
- 如果包含媒体文件，上传过程会在后台进行
- 发布成功后会收到提示

:::warning 发布后修改
帖子发布后暂不支持编辑。如果需要修改内容，可以删除后重新发布。请在发布前仔细检查内容。
:::

## 互动功能

### 点赞 / 取消赞

- 点击帖子下方的「**👍**」按钮即可点赞
- 再次点击取消点赞
- 点赞数实时更新

### 点踩

- 点击「**👎**」按钮对帖子表达不同意
- 再次点击取消点踩
- 你不能同时对一个帖子既点赞又点踩

### 评论

1. 点击帖子下方的「**💬 评论**」按钮
2. 进入评论区
3. 在底部输入框写下评论
4. 点击「发送」

评论功能：

- 每条评论最多 **500 个字符**
- 支持回复他人的评论（嵌套回复）
- 评论按时间正序排列，最早的在上方
- 可以对评论点赞

:::info 评论通知
当有人评论你的帖子或回复你的评论时，你会收到一条通知。可以在通知设置中管理这些提醒。
:::

### 收藏

- 点击「**⭐ 收藏**」按钮将帖子加入收藏夹
- 在「**我的 → 收藏**」中查看所有收藏内容
- 再次点击取消收藏

### 分享

点击「**↗ 分享**」按钮，可以：

- **分享到对话**：将帖子链接发送给某个联系人或群聊
- **复制链接**：复制帖子链接到剪贴板
- **系统分享**：调起系统分享面板，分享至其他 App

## 管理自己的帖子

在帖子详情页或自己的帖子卡片上，点击右上角「**⋮**」可以：

| 操作 | 说明 |
|------|------|
| **删除帖子** | 永久删除帖子及其所有评论（不可恢复） |
| **查看互动数据** | 查看点赞、评论、收藏的详细统计 |

## 举报不当内容

如果你看到不适当的帖子或评论：

1. 点击帖子/评论右上角的「**⋮**」
2. 选择「**举报**」
3. 选择举报原因
4. 提交举报

平台团队会在 24 小时内处理。
