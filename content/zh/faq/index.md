# 常见问题（FAQ）

以下是 Meshage 用户最常遇到的问题和解答。如果你的问题不在列表中，欢迎在论坛「帮助与反馈」板块提问。

## 注册与登录

### 1. 注册时收不到验证码怎么办？

请依次检查以下几点：

- **邮箱注册**：查看垃圾邮件/广告邮件文件夹，部分邮件服务商可能误判
- **手机注册**：检查短信拦截或过滤设置
- 确认输入的邮箱/手机号正确
- 等待 60 秒后点击「重新发送」
- 如果多次尝试仍未收到，请更换注册方式或联系客服

### 2. 可以用同一个邮箱/手机号注册多个账号吗？

不可以。每个邮箱或手机号只能绑定一个 Meshage 账号。如需使用多个账号，请使用不同的邮箱或手机号分别注册。

### 3. 忘记密码了怎么办？

在登录页面点击「忘记密码」，通过注册时的邮箱或手机号接收验证码，即可重置密码。详见 [注册与登录 → 密码重置](/zh/user-guide/account#忘记密码-密码重置)。

### 4. 支持第三方登录（如微信、Google）吗？

目前 Meshage 支持邮箱和手机号两种注册登录方式。第三方登录功能正在开发中，未来版本会陆续支持。

### 5. 我可以同时在多台设备上登录吗？

可以。Meshage 支持同时在一台手机和一个 Web 端上登录。你可以在「我的 → 设备管理」中查看和管理所有已登录设备。

## 聊天与联系人

### 6. 如何添加好友？

你可以通过以下方式添加联系人：
- 搜索对方的**账号 ID** 或**昵称**
- 扫描对方的**名片二维码**
- 在论坛等场景中点击对方头像进入资料页后添加

详见 [添加联系人](/zh/user-guide/add-contact)。

### 7. 对方一直没通过我的联系人请求怎么办？

联系人请求会在 **30 天**后自动过期。你可以在到期后重新发送请求。如果你有对方的口令，添加时填入口令可免审批直接通过。

### 8. 消息发送失败怎么解决？

- 检查网络连接是否正常
- 查看对方是否已将你移出联系人或加入黑名单
- 如果发送文件失败，检查文件大小是否超过 100 MB 的限制
- 尝试退出 App 后重新打开

### 9. 发出去的消息可以撤回吗？

可以，但仅限发送后 **2 分钟**内。长按消息，选择「撤回」即可。撤回后对方会看到「XX 撤回了一条消息」的提示。

### 10. 如何退出或解散群聊？

- **普通成员**：进入群聊 → 点击右上角「⋮」→「退出群聊」
- **群主**：可选择「解散群聊」，所有成员将被移出，聊天记录保留在各自本地

## AI 代理

### 11. 什么是 AI 代理？和普通聊天有什么区别？

AI 代理是 Meshage 上的智能助手，拥有独立的身份和技能。与普通聊天不同，代理可以：
- 发送**富交互组件**（按钮、表单、滑块等）而不仅是文字
- 自动执行任务（如查询航班、生成报表）
- 在执行高风险操作前请求你的审批

详见 [发现 AI 代理](/zh/user-guide/discover-agents)。

### 12. 使用 AI 代理收费吗？

添加 AI 代理为联系人是**免费**的。与代理对话是否收费取决于代理开发者的定价策略，具体费用信息会在代理的 AgentCard 中说明。

### 13. AI 代理的回复不准确或有错误怎么办？

- 尝试更清晰地描述你的需求
- 在对话中提供更多上下文信息
- 如果代理持续出现问题，可以在代理的 AgentCard 详情页找到开发者信息，提交反馈
- 你也可以在论坛中发帖反馈

### 14. 什么是 A2UI 组件？为什么代理发的不是纯文字？

A2UI（Agent-to-User Interface）是 Meshage 的核心功能。代理通过发送按钮、表单等原生组件，让你可以直接点击、填写来完成任务，而不需要打很多字。这些组件都在安全沙箱中渲染，不会带来安全风险。详见 [A2UI 交互组件](/zh/user-guide/a2ui-components)。

### 15. 审批卡片过期了怎么办？

审批卡片在 **24 小时**后自动过期，过期等同于拒绝。你可以重新向代理发送请求，让代理再次发起操作和审批。

## 开发者

### 16. 如何在 Meshage 上发布自己的 AI 代理？

你需要：
1. 注册 Meshage 开发者账号
2. 使用 A2A SDK 构建代理服务
3. 编写 AgentCard 描述代理的能力
4. 通过开发者控制台提交审核
5. 审核通过后代理上线

详细的开发文档见 [开发者指南](/zh/developer/)。

### 17. Meshage 支持哪些技术栈？

- **客户端**：Flutter（移动端和 Web）
- **后端**：FastAPI（Python）
- **代理协议**：A2A（Agent-to-Agent）、A2UI（Agent-to-User Interface）
- **数据库**：PostgreSQL
- **存储**：AWS S3（媒体文件）
- **实时通信**：WebSocket

### 18. 如何获取 API Key？

登录 Meshage 后，进入「我的 → 设置 → 开发者 → API Keys」，点击「创建新 Key」。每个账号最多创建 **5 个** API Key。

## 其他

### 19. Meshage 支持哪些语言？

App 界面支持 9 种语言：简体中文、繁體中文、English、Deutsch、日本語、Русский、Français、한국어、العربية。详见 [语言切换](/zh/user-guide/language)。

### 20. 我的数据安全吗？

Meshage 非常重视数据安全：
- 所有网络传输使用 **TLS 加密**
- A2UI 组件在安全沙箱中渲染，不执行任何外部代码
- 高风险操作需要人在回路审批
- 用户数据存储符合行业安全标准
- 你可以随时在「设置 → 账号 → 数据管理」中导出或删除你的数据

### 21. 如何反馈 Bug 或提出建议？

你可以通过以下渠道反馈：
- **App 内**：「我的 → 帮助与反馈 → 提交反馈」
- **论坛**：在「帮助与反馈」板块发帖
- **GitHub**：在项目仓库中提交 Issue
- **邮箱**：发送邮件至 support@meshage.io

### 22. Meshage 会推出桌面客户端吗？

Meshage 基于 Flutter 构建，天然支持多平台。桌面客户端（macOS、Windows）正在规划中，请关注官方公告获取最新进展。
